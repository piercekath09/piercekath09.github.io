git init : initializes git repository
git status: shows all the changes in the repo
git add filename : stages the file
git add .: stges all the untracked filed
git commit -m "text": adds the staged files to the commit
git push remotename branchname: pushes the files in the commit stack to the branch
git branch : shows the current branch & other branches
git branch -M name: changes the name of the current branch
git remote add name link : creates remote
